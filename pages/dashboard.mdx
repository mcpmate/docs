## Resource Usage

Monitor CPU and memory consumption to ensure optimal performance.

- **CPU Usage**: Real-time processor utilization by MCPMate and connected servers
- **Memory Usage**: RAM consumption with breakdown by component
- **Network**: Data transfer rates for MCP communications

<Callout type="info">
  Resource metrics update every 10 seconds. High usage may indicate server configuration issues.
</Callout>

The logs panel shows:
- Server startup/shutdown events
- Tool execution requests and responses  
- Client app connection status
- Configuration changes
- Error messages and warnings

### Log Filtering

Use the filter controls to:
- Show logs from specific servers or suits
- Filter by log level (Info, Warning, Error)
- Search for specific messages or timestamps
- Export logs for troubleshooting

## Quick Actions

From the Dashboard, you can:
- Toggle configuration suits on/off
- Restart problematic servers
- View detailed server information
- Access system settings
