## Download and Install

<AccordionGroup>
  <Accordion icon="download" title="Download MCPMate">
    1. Visit [mcpmate.io](https://mcpmate.io) and click **Download for macOS**.
    2. Save the `.dmg` file to your Downloads folder.
    3. Double-click the `.dmg` to mount it, then drag MCPMate to your Applications folder.
  </Accordion>
  
  <Accordion icon="rocket" title="First Launch">
    1. Open MCPMate from Applications (you may need to right-click â†’ Open if macOS shows a security warning).
    2. MCPMate will automatically scan for MCP-compatible apps like Cursor, Claude Desktop, and Zed.
    3. If found, you'll see them listed under **Client Apps** with toggle switches.
  </Accordion>
</AccordionGroup>

## Quick Setup

<CardGroup cols={2}>
  <Card title="Enable Default Config" icon="toggle-on">
    MCPMate ships with a **Default** configuration suit. Simply toggle it on to start using basic MCP tools.
  </Card>
  
  <Card title="Test in Your Editor" icon="code">
    Open Cursor or Claude Desktop and try an MCP-powered command. Tools should appear automatically.
  </Card>
</CardGroup>
