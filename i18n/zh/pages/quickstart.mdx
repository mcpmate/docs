## 下载和安装

<AccordionGroup>
  <Accordion icon="download" title="下载 MCPMate">
    1. 访问 [mcpmate.io](https://mcpmate.io) 并点击 **下载 for macOS**。
    2. 将 `.dmg` 文件保存到您的下载文件夹。
    3. 双击 `.dmg` 文件挂载它，然后将 MCPMate 拖动到您的应用程序文件夹。
  </Accordion>
  
  <Accordion icon="rocket" title="首次启动">
    1. 从应用程序打开 MCPMate（如果 macOS 显示安全警告，您可能需要右键点击 → 打开）。
    2. MCPMate 将自动扫描 MCP 兼容的应用程序，如 Cursor、Claude Desktop 和 Zed。
    3. 如果找到，您将看到它们列在 **客户端应用** 下，带有切换开关。
  </Accordion>
</AccordionGroup>

## 快速设置

<CardGroup cols={2}>
  <Card title="启用默认配置" icon="toggle-on">
    MCPMate 随附一个 **默认** 配置。只需切换它即可开始使用基本的 MCP 工具。
  </Card>
  
  <Card title="在您的编辑器中测试" icon="code">
    打开 Cursor 或 Claude Desktop 并尝试一个 MCP 功能。工具应该会自动出现。
  </Card>
</CardGroup>
