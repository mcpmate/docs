## 管理连接

对于每个检测到的应用程序，您可以：

- **启用/禁用**: 开启或关闭 MCP 集成
- **查看状态**: 查看连接健康状况和最后活动时间
- **配置设置**: 设置应用程序特定的代理设置
- **重定向**: 选择要使用的配置套件

<Callout type="info">
  在 MCPMate 中启用/禁用 MCP 集成后，必须重启应用程序。
</Callout>

## 手动配置

如果您的应用程序未被自动检测到：

1. 点击 **添加自定义应用**
2. 指定应用程序名称和 MCP 配置方法
3. 选择连接类型（stdio、SSE 或 HTTP）
4. 测试连接

## 连接类型

MCPMate 支持多种 MCP 连接方法：

- **stdio**: 直接进程通信（最常见）
- **SSE**: Server-Sent Events over HTTP
- **HTTP**: RESTful API communication

## 故障排除

常见问题和解决方案：

- **应用未检测到**: 确保应用安装在标准位置
- **连接失败**: 检查应用权限和防火墙设置
- **工具未显示**: 验证应用是否支持您的活动配置套件
